# Marionette boilerplate

## Just what i(you) need to start my(your) next Marionette application.

### What i use :

+ backbone 1.1.2
+ marionette 2.3.1
+ lodash 3.0.0
+ jQuery 2.1.3
+ requirejs 2.1.15
+ with text plugin 2.0.13
+ hogan 3.0.2

### Make it works :

npm install && bower install

### Tasks :

+ grunt serve
+ grunt test
+ grunt dist
