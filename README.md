# Marionette boilerplate

## Just what i(you) need to start my(your) next Marionette application.

### What i use :

+ backbone 1.0.0
+ marionette 1.0.4
+ underscore 1.4.4
+ jQuery 2.0.3
+ requirejs 2.1.6
+ with text plugin 2.0.5+
+ mustache 0.7.2
+ hogan 3.0.0
+ Twitter Boostrap 3.0.0-rc1
+ less 1.3.3

### Make it works :

npm install
and then go to public folder
and then : bower install
and then it's ok :)
