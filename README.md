# Marionette boilerplate

## Just what i(you) need to start my(your) next Marionette application.

### What i use :

+ backbone 1.1.2
+ marionette 1.6.2
+ underscore 1.6.0
+ jQuery 2.1.0
+ requirejs 2.1.11
+ with text plugin 2.0.10
+ mustache 0.8.1
+ hogan 3.0.0
+ sass 3.2.12

### Make it works :

npm install
and then bower install
and then install sass/bourdon/neat
and then it's ok :)
