# Marionette boilerplate

## Just what i(you) need to start my(your) next Marionette application.

### What i use :

+ backbone 1.1.2
+ marionette 2.1.0
+ underscore 1.7.0
+ jQuery 2.1.1
+ requirejs 2.1.14
+ with text plugin 2.0.12
+ mustache 0.8.2
+ hogan 3.0.2
+ sass 3.4.2

### Make it works :

npm install && bower install

### Tasks :

+ grunt serve
+ grunt test
+ grunt dist
